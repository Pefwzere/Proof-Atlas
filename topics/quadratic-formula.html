Quadratic Formula

The formula for solving equations of the form ax² + bx + c = 0, where a is not equal to zero.

Formula

x = -b ± b² - 4ac 2a

This gives the possible values of x for any quadratic equation written as ax² + bx + c = 0.

Meaning of the Symbols

  • a is the coefficient of x².
  • b is the coefficient of x.
  • c is the constant term.
  • ± means the equation may have two answers: one using plus and one using minus.

Derivation by Completing the Square

We begin from the general quadratic equation.

ax² + bx + c = 0

First, move the constant term to the other side.

ax² + bx = -c

Divide every term by a, because the coefficient of x² must be 1 before completing the square.

x² + bax = -ca

Take half of the coefficient of x, then square it. Half of b/a is b/2a, so we add (b/2a)² to both sides.

x² + bax + (b2a)² = -ca + (b2a)²

The left side is now a perfect square.

(x + b2a)² = -ca + b²4a²

Write the right side over the same denominator, 4a².

(x + b2a)² = -4ac + b²4a² (x + b2a)² = b² - 4ac4a²

Take the square root of both sides. This is where the plus or minus sign enters.

x + b2a = ± b² - 4ac2a

Finally, move b/2a to the other side and combine the terms into one fraction.

x = -b2a ± b² - 4ac2a x = -b ± b² - 4ac2a

Root Relationship Method

Another way to understand a quadratic is to call its two roots α and β. If α and β are the roots, then the quadratic can be written from its factors.

a(x - α)(x - β) = 0

Expanding the bracket gives:

a[x² - (α + β)x + αβ] = 0 ax² - a(α + β)x + aαβ = 0

Compare this with ax² + bx + c = 0.

α + β = -ba αβ = ca

So the sum of the roots is -b/a, and the product of the roots is c/a. This method helps explain how the roots behave, while completing the square gives the full formula for finding them.

Example

Solve x² - 5x + 6 = 0.

a = 1, b = -5, c = 6

Substitute the values into the formula.

x = 5 ± 25 - 242
x = 5 ± 12

So x = 3 or x = 2.

Notes

  • The expression b² - 4ac is called the discriminant.
  • If b² - 4ac is positive, there are two real roots.
  • If b² - 4ac is zero, there is one repeated real root.
  • If b² - 4ac is negative, there are no real roots.
